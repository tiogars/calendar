# 03.01 - Screenshots

Place screen captures in this folder.

## Screenshot Files

| File | Viewport | Description |
| ---- | -------- | ----------- |
| `edition-page.png` | 1280 × 900 | Calendar settings form (desktop) |
| `preview-page.png` | 1280 × 900 | A4 calendar preview page (desktop) |
| `mobile-edition-page.png` | 390 × 844 | Edition form on mobile with FAB |
| `mobile-preview-page.png` | 390 × 844 | Scaled A4 preview on mobile |
| `print-result.png` | — | Optional: browser print-preview dialog |

## Notes

- Use PNG format for best readability in documentation.
- Prefer full-window captures (not cropped) at the stated viewport size.
- Keep captures up to date when UI or features change.
- Desktop captures use 1280 × 900 pixels.
- Mobile captures use 390 × 844 pixels (iPhone 14 equivalent).
